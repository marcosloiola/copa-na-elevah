🏆 Gerenciar Álbuns

Crie até 3 álbuns (um por pessoa, por exemplo). O álbum ativo é o que está sendo editado nas abas Lista e Faltam. Você pode renomear ou excluir álbuns, mas não é possível excluir o único álbum.

🔄 Sincronização

Transfere todos os seus álbuns entre dispositivos. O código é gerado automaticamente — copie e cole via WhatsApp no outro celular, depois clique em Aplicar Código.

🤝 Troca de Figurinhas

Via QR Code (mais rápido): Toque em "Gerar meu QR Code". Mostre para o amigo ler com a câmera nativa do celular dele — o app vai abrir automaticamente e mostrar quais repetidas você tem que ele precisa. Não é necessário nenhum leitor dentro do app.

Via código de texto: Gere o código de texto e envie por WhatsApp. O amigo cola o código e vê as coincidências. Funciona offline após o primeiro acesso.

📋 Exportar Listas

Gera listas de texto das figurinhas faltantes ou repetidas para compartilhar via WhatsApp ou copiar. A lista de faltantes combina todos os álbuns e indica com (×N) quantos precisam de cada figurinha.

⚠️ Zona de Perigo

Permite apagar as figurinhas de um álbum específico, de todos os álbuns de uma vez, ou limpar a lista de repetidas. Os álbuns em si não são excluídos ao apagar as figurinhas — apenas as marcações são removidas.

📤 📥 Exportar e Importar álbum individual

Cada álbum tem botões próprios de exportar (📤) e importar (📥) na lista de álbuns. Exportar gera um código só daquele álbum. Importar aplica um código e substitui as figurinhas marcadas. Use para sincronizar um álbum específico, incluindo o fluxo de troca com amigo.

👥 Álbum de Amigo

Você pode importar o álbum de um único amigo (slot extra além dos seus 3 álbuns). O álbum aparece em Faltam e Visão Geral com visual azul, separado dos seus álbuns.

Fluxo completo de troca

  • Amigo gera o código: ele vai em Gerenciar Álbuns → 📤 no álbum dele → envia o código pra você
  • Você importa: Gerenciar Álbuns → "Importar Álbum do Amigo" → cola o código
  • Você marca: na aba Faltam, selecione o pill azul do amigo → marque as figurinhas que você conseguiu pra ele
  • Você devolve: Gerenciar Álbuns → 📤 no card do amigo → envia o código de volta
  • Amigo aplica: ele usa o botão 📥 no álbum dele para atualizar com o código que você mandou

📊 Visão Geral

Mostra o progresso do álbum. Se você tiver mais de um álbum, exibe um card por álbum (toque para ver detalhes por seleção) e um card "Total" com a soma de todos. Com apenas um álbum, abre diretamente no detalhamento.

📱 Instalar como app

iPhone (Safari): botão compartilhar ↑ → Adicionar à Tela de Início.

Android (Chrome): menu ⋮ → Adicionar à tela inicial.

💾 Onde ficam os dados?

Tudo salvo no localStorage do navegador — no dispositivo, sem depender de internet. Para transferir, use o Código de Sincronização.

🗑️ Apagar figurinhas do álbum?

Vai remover todas as figurinhas marcadas do álbum ativo. O álbum em si não será excluído. Esta ação não pode ser desfeita.

📤 Exportar Álbum

Compartilhe este código com seu amigo para que ele importe como álbum de amigo. Quando ele devolver o código atualizado, use o botão 📥 para aplicar.

📥 Importar para Álbum

Cole o código do álbum. As figurinhas marcadas serão substituídas pelas do código.

👥 Importar Álbum do Amigo

Cole o código que seu amigo gerou pelo botão 📤 no app dele.

📤 Devolver Código ao Amigo

Envie este código para seu amigo. Ele pode usar o botão 📥 no álbum dele para atualizar as figurinhas.

🗑️ Apagar figurinhas de todos os álbuns?

Vai remover todas as figurinhas marcadas de todos os seus álbuns. Os álbuns em si não serão excluídos. Esta ação não pode ser desfeita.

🗑️ Apagar Lista de Repetidas?

Remove todas as marcações de repetidas. Esta ação não pode ser desfeita.

➕ Criar Novo Álbum

Dê um nome para identificar este álbum (ex: "Álbum da Maria").

🗑️ Excluir álbum?

Vai excluir permanentemente o álbum e todas as suas figurinhas marcadas.
