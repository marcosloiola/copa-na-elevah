🎴 O que é este app?

Gestor do álbum de figurinhas da Copa do Mundo 2026, criado pela Elevah. Controle quais figurinhas você tem, quais faltam e quais estão repetidas — tudo no seu celular ou computador.

👆 Como marcar figurinhas

As figurinhas aparecem como botões numerados nas abas Lista, Faltantes e Repetidas:

7
Não tenho
7
Tenho ✓
7
Repetida
  • 1 toque → marca que você tem a figurinha (ou que ela está repetida, na aba Repetidas)
  • 2 toques rápidos → desfaz a marcação
  • Todas as abas atualizam automaticamente após cada marcação

📊 Visão Geral

A primeira aba mostra gráfico de progresso geral e por grupo/seleção. Atualiza em tempo real.

🔄 Sincronizar entre dispositivos

Na aba Configurações → Sincronizar:

  • Gerar Código: clique e um código de texto será gerado com todo o seu progresso
  • Clique em Copiar e mande pelo WhatsApp para o outro dispositivo
  • No outro dispositivo, cole o código em Importar via Código e clique em Aplicar

🤝 Troca de Figurinhas

  • Gerar Código das Minhas Faltantes: gera um código com as figurinhas que você ainda precisa. Envie para um amigo.
  • O amigo cola esse código em Verificar Código de um Amigo → o app mostra quais das repetidas dele você precisa — facilitando a troca!

💾 Onde ficam os dados?

Tudo é salvo no localStorage do seu navegador — fica no dispositivo. Para transferir para outro aparelho, use o Código de Sincronização.

🗑️ Apagar Álbum Completo?

Isso vai remover todas as figurinhas marcadas como "tenho". Esta ação não pode ser desfeita.

🗑️ Apagar Lista de Repetidas?

Isso vai remover todas as marcações de repetidas. Esta ação não pode ser desfeita.
